# Privacy Policy for Poker Chip Manager

**Last Updated: [08/11/25]**

## Overview

Poker Chip Manager ("we," "our," or "the app") is committed to protecting your privacy. This Privacy Policy explains how we collect, use, and safeguard your information when you use our mobile application.

---

## Information We Collect

### 1. Information You Provide
- **Nickname**: When you join a game, you provide a nickname for identification within game rooms
- **Game Data**: Your chip counts, bets, and game actions within poker rooms

### 2. Automatically Collected Information
- **Device Information**: Device ID for anonymous authentication
- **Usage Data**: App interactions, features used, and session duration
- **Connection Status**: Online/offline status for game management

### 3. Information We Do NOT Collect
- ❌ No email addresses
- ❌ No phone numbers
- ❌ No real names
- ❌ No payment information
- ❌ No location data
- ❌ No personal identifiable information (PII)

---

## How We Use Your Information

We use the collected information to:

1. **Provide Game Functionality**
   - Enable multiplayer poker game rooms
   - Track chips and bets in real-time
   - Manage turn-based gameplay

2. **Improve User Experience**
   - Maintain game state across sessions
   - Provide seamless reconnection
   - Display accurate game logs

3. **App Performance**
   - Monitor app crashes and errors
   - Analyze usage patterns to improve features
   - Optimize performance

---

## Data Storage and Security

### Storage
- All game data is stored securely using **Google Firebase**
- Data is encrypted in transit using industry-standard HTTPS
- Game rooms are automatically deleted when all players leave

### Security Measures
- Anonymous authentication (no personal credentials required)
- Secure database rules prevent unauthorized access
- Regular security updates and monitoring

### Data Retention
- **Active Game Data**: Stored while a game room is active
- **Auto-Deletion**: Game rooms automatically delete when empty
- **No Long-Term Storage**: We do not store historical game data

## Children's Privacy

Poker Chip Manager does not knowingly collect information from children under 13. The app is intended for general audiences. If you believe a child under 13 has provided information to us, please contact us immediately.

## Your Rights and Choices

### Data Access
- You can view your nickname and chip count within the app
- Game data is visible to all players in your room

### Data Deletion
- Leave the room to remove your data from that game
- Uninstall the app to remove local data
- Contact us to request deletion of any residual data

### Opt-Out Options
- Don't provide a nickname (though this limits functionality)
- Don't join game rooms to avoid data collection
